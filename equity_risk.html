1a. Interpretation of Model 1a
The estimated model is:
rNVDA = 0 + 1 rSNP500 + 2 rGold + u


The intercept estimate is 0.022 with a p value of 0.406. It is not statistically different from zero, which indicates that NVDAs average return is essentially zero when both explanatory variables are zero.
The coefficient on the market return is 0.048 with a p value of 0.074. This estimate is small and only marginally significant. It suggests that NVDAs sensitivity to market movements is limited in this specification.
The coefficient on gold returns is 0.291 with a p value below 0.001. Gold returns are therefore a statistically significant explanatory variable.


A one percent increase in gold returns is associated with a 0.29 percent increase in NVDA returns, while a one percent increase in market returns corresponds to only a 0.05 percent increase. The signs and magnitudes indicate that gold contributes meaningfully to NVDAs short-run variation, while market sensitivity remains limited in this specification.


The model explains approximately 50.7 percent of the variation in NVDAs daily returns. Residual diagnostics indicate non normality, skewness, heavy tails, and the presence of outliers, which are common in financial return data.


This model captures a substantial part of return variation, although the presence of volatility spikes indicates that additional dynamics such as heteroscedasticity may still be relevant.

# Question 2: Test if 1 = 1 (market beta equals 1)
# H0: 1 = 1
# H1: 1  1

print("="*70)
print("QUESTION 2: HYPOTHESIS TEST FOR  = 1")
print("="*70)
print("H:  = 1")
print("H:   1")
print("\n")

# Extract coefficient and standard error for 1 (rSNP500)
beta1_estimate = model_1a.params['rSNP500']
beta1_se = model_1a.bse['rSNP500']

# Calculate t-statistic for H0: 1 = 1
t_statistic = (beta1_estimate - 1) / beta1_se

# Degrees of freedom
df = model_1a.df_resid

# Calculate p-value (two-tailed test)
p_value = 2 * (1 - stats.t.cdf(abs(t_statistic), df))

# Critical value at 5% significance level (two-tailed)
alpha = 0.05
t_critical = stats.t.ppf(1 - alpha/2, df)

print(f"Estimated :        {beta1_estimate:.6f}")
print(f"Standard Error:      {beta1_se:.6f}")
print(f"Hypothesized Value:  1.0000")
print(f"\nTest Statistic (t):  {t_statistic:.6f}")
print(f"Degrees of Freedom:  {df}")
print(f"P-value:             {p_value:.6f}")
print(f"Critical Value (5%): {t_critical:.6f}")

# Decision
print("\n" + "-"*70)
print("DECISION:")
print("-"*70)
if p_value < alpha:
    print(f" REJECT H (p-value = {p_value:.6f} < 0.05)")
    print(f"  There is sufficient evidence that   1")
else:
    print(f" FAIL TO REJECT H (p-value = {p_value:.6f} > 0.05)")
    print(f"  There is insufficient evidence that   1")

# Confidence interval
ci_lower = beta1_estimate - t_critical * beta1_se
ci_upper = beta1_estimate + t_critical * beta1_se
print(f"\n95% Confidence Interval for : [{ci_lower:.6f}, {ci_upper:.6f}]")
print(f"Does the CI contain 1? {'YES' if ci_lower <= 1 <= ci_upper else 'NO'}")
